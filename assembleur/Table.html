Table des instructions

Il s'agit de la liste des instructions qui peuvent être exécutées sur la machine pédagogique. L'utilisateur ne doit pas utiliser un nom d'instruction qui ne figure pas sur cette liste.

Notations:

  • X : c'est une adresse quelconque en mémoire centrale (comprise entre 800 et 999).
  • [ α ] : Contenu de α (α peut être une adresse ou un registre)
  • (*) : signifie Mode d'adressage (voir plus bas).
Instruction (*) Signification
0 LOAD X, (*) D Acc ← [X]
IND Acc ← [[X]]
IMM Acc ← X
XRi Acc ← [ X + [XRi] ]
1 STORE X, (*) D @X ← [Acc]
IND @[X] ← [Acc]
IMM Interdit
XRi @[X + [XRi] ] ← [Acc]
2 ADD X, (*) D Acc ← [Acc] + [X]
IND Acc ← [Acc] + [ [X] ]
IMM Acc ← [Acc] + X
XRi Acc ← [Acc] + [ X + [XRi] ]
3 SUB X, (*) D Acc ← [Acc] - [X]
IND Acc ← [Acc] - [ [X] ]
IMM Acc ← [Acc] - X
XRi Acc ← [Acc] - [ X + [XRi] ]
4 MUL X, (*) D Acc ← [Acc] * [X]
IND Acc ← [Acc] * [ [X] ]
IMM Acc ← [Acc] * X
XRi Acc ← [Acc] * [ X + [XRi] ]
5 DIV X, (*) D Acc ← [Acc] / [X]
IND Acc ← [Acc] / [ [X] ]
IMM Acc ← [Acc] / X
XRi Acc ← [Acc] / [ X + [XRi] ]
6 AND X, (*) D Acc ← [Acc] and [X]
IND Acc ← [Acc] and [ [X] ]
IMM Acc ← [Acc] and X
XRi Acc ← [Acc] and [ X + [XRi] ]
7 OR X, (*) D Acc ← [Acc] or [X]
IND Acc ← [Acc] or [ [X] ]
IMM Acc ← [Acc] or X
XRi Acc ← [Acc] or [ X + [XRi] ]
8 NOT X, (*) D Acc ← not [X]
IND Acc ← not [ [X] ]
IMM Acc ← not X
XRi Acc ← not [ X + [XRi] ]
9 JMP X, (*) D MAR ← [X]
IND MAR ← [ [X] ]
IMM MAR ← X
XRi MAR ← [ X + [XRi] ]
10 JE   CO ← [CO] +1 si [Acc] = 0
11 JG   CO ← [CO] +1 si [Acc] > 0
12 JL   CO ← [CO] +1 si [Acc] < 0
13 MOVE XRi   XRi ← [ ACC ]
14 READ   Acc ← [ Clavier ]
15 WRITE   Ecran ← [ Acc ]
